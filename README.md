> Window Swallowing
Refer to this [Tutorial](https://www.youtube.com/watch?v=iB4aBY0H_oI) for a good guide by a euroman

**tl:dr**
map your programs to `dwmswallow $WINDOWID; *Program Name*` in your .bashrc
